# 007-Everything-or-Nothing-Autosplitter
ASL and Blank Splits for autosplitting the GameCube version of 007 Everything or Nothing on Dolphin emulators. Will work on versions of Dolphin from 5.0 onwards.

ASL Includes:
* Start and Split - Reset not supported
* Option to choose whether to split upon selecting a new level, or the end of your current level
* Option to include Boss Fight and Leap of Faith mid-level splits

Requires emu-help-v3 to be added to your LiveSplit\Components folder: https://github.com/Jujstme/emu-help-v3

Also available are blank split files for both all levels, and all leves + mid-level splits.
